# Social Media and Mental Health Analysis

## Objective
Exploratory data analysis on how social media usage affects mental health

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Research Questions
1. Does higher screen time lead to higher stress levels?
2. Which platform has the highest average stress level?
3. Does screen time affect sleep quality?
4. Which age group experiences the highest stress level?
5. Do stress levels differ across genders?
6. Which platform users report the worst sleep quality?
7. Which platform-age group combination shows the highest stress level?

## Key Finding
Instagram users in the Young Adult age group reported the highest average stress level (7.82).
